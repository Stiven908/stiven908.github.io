Descripción de proyecto

El proyecto final del curso de Sistemas Operativos tiene como propósito integrar y aplicar los conocimientos adquiridos durante la asignatura mediante el desarrollo de actividades prácticas enfocadas en el uso, administración y despliegue de tecnologías basadas en Linux. A lo largo de ocho actividades, se instaló y configuro distribuciones de Linux tanto en entorno gráfico como en consola, además de, gestionar editores de texto, crear scripts de automatización, administrar procesos del sistema y desarrollar programas concurrentes en Java. Además, el proyecto incluye la implementación de Docker Engine, la creación de imágenes personalizadas, el despliegue de contenedores y su publicación en Docker Hub.

Posteriormente, se deberá instalar y configurar un entorno de Kubernetes (Minikube o K3s) para desplegar aplicaciones usando la imagen Docker creada, reforzando así los conocimientos de orquestación de contenedores. Finalmente, el proyecto culmina con la creación de un sitio web que recopila y organiza todos los entregables (videos, capturas, Dockerfiles, scripts, código, etc.).

Objetivos

El objetivo de este proyecto es profundizar y aplicar conocimientos en sistemas operativos mediante la instalación, configuración y administración de distribuciones Linux, el desarrollo de scripts y programas concurrentes, y la implementación de tecnologías de virtualización y contenedores como Docker y Kubernetes. A través de estas actividades, se desarrollarán competencias prácticas esenciales en automatización, gestión de procesos, despliegue de aplicaciones y orquestación de contenedores, reforzando formación teórica y preparando así, para desafíos profesionales en entornos tecnológicos reales.

Objetivos especificos

  • Realizar la instalación y configuración de distribuciones Linux en entornos gráfico y consola.
  • Instalar y personalizar editores de texto y entornos de desarrollo.
  • Diseñar y desarrollar scripts de automatización en Bash.
  • Demostrar competencias en la gestión de procesos en Linux.
  • Instalar y configurar Docker Engine en Linux
  • Modificar y optimizar imágenes Docker
  • Instalar y configurar un entorno Kubernetes (Minikube o K3s)
